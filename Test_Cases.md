# Test Cases

## Login Functionality

### TC-01: Valid login
Steps:
1. Open login page
2. Enter valid username
3. Enter valid password
4. Click Login

Expected Result:
- User is logged in successfully

---

### TC-02: Invalid password
Steps:
1. Open login page
2. Enter valid username
3. Enter invalid password
4. Click Login

Expected Result:
- Error message is displayed

---

### TC-03: Empty fields
Steps:
1. Open login page
2. Leave username and password empty
3. Click Login

Expected Result:
- Validation message is shown
